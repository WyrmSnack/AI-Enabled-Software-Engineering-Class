# Capstone Project Ideas

A curated list of project ideas for the capstone assignment, organized by difficulty level.

## Beginner Projects

### 1. Personal Task Manager
**Problem**: Need a simple way to track personal tasks and to-dos.

**Tech Stack**: FastAPI, React, SQLite/PostgreSQL

**Features**:
- Create, read, update, delete tasks
- Mark tasks as complete
- Filter by status
- Basic authentication

**Learning Objectives**:
- CRUD operations
- Basic authentication
- Simple state management
- Form handling

**Scope**: 3 days achievable

---

### 2. Book Collection Tracker
**Problem**: Track personal book collection, reading progress, and reviews.

**Tech Stack**: FastAPI, React, PostgreSQL

**Features**:
- Add books with details (title, author, ISBN)
- Track reading status (want to read, reading, completed)
- Add ratings and reviews
- Search and filter books

**Learning Objectives**:
- Database relationships
- Search functionality
- Data validation
- API filtering

**Scope**: 3 days achievable

---

## Project Selection Criteria

### Choose a project that:
1. **Solves a real problem** (even if simple)
2. **Fits the 3-day timeline** (MVP scope)
3. **Demonstrates key concepts**:
   - CRUD operations
   - Authentication
   - Database relationships
   - API design
   - Frontend-backend integration
4. **Allows AI tool usage** across SDLC phases
5. **Is personally interesting** to you

### Avoid projects that:
- Are too complex for 3 days
- Require external APIs (unless simple)
- Need real-time features (unless you have experience)
- Require complex algorithms
- Need mobile apps (unless web-first)

---

## Project Planning Tips

1. **Start with MVP**: Identify core features that make the app functional
2. **Prioritize features**: Must-have vs. nice-to-have
3. **Use AI for scaffolding**: Generate initial structure with AI
4. **Iterate quickly**: Get something working, then enhance
5. **Document as you go**: Use AI to generate documentation
