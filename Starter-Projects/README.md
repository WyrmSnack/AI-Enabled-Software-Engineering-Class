# Starter Projects

This folder contains working starter projects that students can use as templates or reference.

## Available Starters

### 1. FastAPI + React Full-Stack Starter
A complete, deployable full-stack application with:
- FastAPI backend with SQLAlchemy
- React frontend with Vite
- PostgreSQL database
- Docker setup
- CI/CD with GitHub Actions
- Complete authentication
- Example CRUD operations

**Location**: `fastapi-react-starter/`

**Quick Start**:
```bash
cd fastapi-react-starter
docker-compose up -d
# Backend: http://localhost:8000
# Frontend: http://localhost:5173
```

### 2. Minimal FastAPI API
A minimal FastAPI setup for quick prototyping:
- Basic FastAPI structure
- SQLite database
- Simple CRUD endpoints
- Basic error handling

**Location**: `minimal-fastapi/`

## Using These Starters

1. **Copy the starter** to your project directory
2. **Customize** for your needs
3. **Add your features** using AI assistance
4. **Deploy** when ready

## Notes

- All starters are working and tested
- Environment variables are documented in `.env.example`
- Dependencies are pinned in `requirements.txt` or `package.json`
- README files in each starter provide setup instructions
